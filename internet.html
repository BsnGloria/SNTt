Qu’est-ce qu’Internet ?

Internet est un réseau mondial d’ordinateurs connectés entre eux. Il permet d’échanger des informations, de communiquer, de partager des fichiers, et d’accéder à des services comme le Web, les mails ou les réseaux sociaux.

Comment ça fonctionne ?

  • Les appareils sont connectés via des réseaux (fibre, Wi-Fi, 4G...)
  • Les données sont envoyées en paquets
  • Les protocoles comme TCP/IP assurent la bonne transmission

Adresses IP et DNS

Chaque appareil possède une adresse IP, comme une adresse postale. Les DNS traduisent les noms de sites (comme google.com) en adresses IP pour qu’on puisse les retrouver facilement.

Internet ≠ Web

Internet est le réseau. Le Web est un service d’Internet qui permet d’accéder à des pages avec un navigateur (comme Chrome ou Firefox).

Exemples de services sur Internet

  • Web (sites, blogs)
  • Emails
  • Streaming (YouTube, Netflix)
  • Réseaux sociaux (Instagram, TikTok)
  • Jeux en ligne
